Program dibuat menggunakan bahasa C# dengan IDE VSCode pada Windows. Untuk instalasi dan menjalankan program pada VSCode, baca lebih lanjut pada `https://code.visualstudio.com/docs/languages/csharp`

## Direktori
```.
├───doc
├───src
│   ├───bin
└───test
```
### Penjelasan Tiap Direktori
1. `bin`: menyimpan file executable 
2. `doc`: dokumentasi dan laporan akhir
3. `src`: menyimpan kode program
4. `test`: menyimpan kasus uji

## Kompilasi Program
pada root, gunakan perintah
```bash
dotnet build
dotnet run --project src
```
program akan dikompilasi dan dijalankan pada terminal (CLI)

## Menjalankan File Pada Bin
pada /bin gunakan perintah
```bash
.\src\bin\Debug\net10.0\src.exe
```
